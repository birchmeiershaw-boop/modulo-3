# 🛒 E-commerce en Python

Sistema de gestión de productos y carrito de compras desarrollado en Python. Permite navegar por un catálogo de productos, buscar artículos, agregar al carrito y realizar compras simuladas.

---

## 📋 Descripción

Este proyecto es un **sistema de e-commerce de consola** que simula una tienda en línea completa. El usuario puede:

- ✅ Ver el catálogo completo de productos
- 🔍 Buscar productos por nombre o categoría
- ➕ Agregar productos al carrito (con cantidades)
- 👁️ Ver el contenido del carrito y el total
- 🗑️ Eliminar productos específicos o vaciar el carrito
- 💳 Finalizar la compra con resumen

El sistema está organizado en módulos para facilitar el mantenimiento y escalabilidad del código.
